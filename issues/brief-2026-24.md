## Enterprise & security implications
- **Semantic Kernel agent-RCE now has CVEs + shipped patches (resolves wk23 watch item).**
  Microsoft assigned **CVE-2026-26030** (In-Memory Vector Store default filter builds a Python
  lambda and runs it via `eval()` → host RCE) and **CVE-2026-25592** (.NET SDK
  `SessionsPythonPlugin` exposed a host-side file-download method as a callable kernel function).
  MSRC advisories + "When prompts become shells" research post published **May 7, 2026**. Patched
  in **semantic-kernel ≥1.39.4 (Python)** and **≥1.71.0 (.NET)**; fix adds AST allowlist +
  function-call allowlist + dangerous-attribute blocklist. Microsoft Security Blog is PRIMARY.
  - https://www.microsoft.com/en-us/security/blog/2026/05/07/prompts-become-shells-rce-vulnerabilities-ai-agent-frameworks/
  - https://www.pointguardai.com/ai-security-incidents/semantic-kernel-lets-a-prompt-open-a-shell-cve-2026-25592-cve-2026-26030
